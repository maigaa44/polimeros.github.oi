APLICACIONES DEL PU

El poliuretano es un polímero que se usa en muchas aplicaciones, entre ellas: Espuma de poliuretano: Se usa en la fabricación de muebles, ropa de cama, bases para alfombras, interiores de automóviles y embalajes. Aislamiento térmico: Se usa en paredes, suelos, cubiertas, medianeras y bajo forjados. Pinturas: Se usa para pintar puertas y carrocerías. Pegamentos: Se usa para unir materiales como madera, cemento, gomas, vidrios o cartón. Tablas de surf: Se usa para fabricar tablas de surf que sean ligeras y resistentes a los golpes de las olas. Gasa médica: Se usa en forma de gasa para reemplazar al algodón en la labor médica diaria. Prótesis mamarias: Se usa para recubrir prótesis mamarias con espuma de poliuretano. Alineadores de ortodoncia: Se usa para fabricar alineadores transparentes y hechos a medida para cada paciente. Implantes quirúrgicos: Se usa en implantes quirúrgicos para aislar el corazón y marcapasos artificial.
