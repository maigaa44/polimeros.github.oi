APLICACIONES DEL PAN

Los polímeros de acrilato tienen muchas aplicaciones, entre ellas: Pinturas: La emulsión de poliacrilato se usa como aglutinante en pinturas de látex para interiores y exteriores. Las pinturas acrílicas también se usan para arte. Fibra acrílica: Se usa en la fabricación de fibra acrílica. Polímero superabsorbente (SAP): El poliacrilato de sodio se usa para producir el SAP, que se utiliza en pañales desechables. Resina acrílica: Se usa como adhesivo sensible a la presión. Cosméticos: Los poliacrilatos se usan como modificadores reológicos y formadores de película en productos cosméticos. Lubricante ocular: El ácido poliacrílico se usa como lubricante del ojo externo en casos de insuficiente secreción lagrimal o daño corneal. Vidrio acrílico: El polimetacrilato de metilo se usa para fabricar vidrio acrílico, también conocido como plexiglás. Construcción: El acrílico se usa en la construcción para hacer domos, cúpulas, lámparas, bañeras y cerramientos de balcones. Publicidad: El acrílico se usa para agregar valor estético a campañas publicitarias. Industria: El acrílico se usa para proteger equipos industriales. Transporte: El acrílico se usa para proporcionar claridad óptica en el transporte.
