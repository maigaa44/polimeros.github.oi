APLICACIONES DEL PS

El poliestireno (PS) tiene muchas aplicaciones, entre ellas: Construcción: Se utiliza en el aislamiento de techos y paredes, como aligerado de losas, azoteas y losas de cimentación. Embalaje: Se utiliza para proteger productos frágiles, como en contenedores isotérmicos. Industria marina: Se utiliza para fabricar flotadores. Industria automotriz: Se utiliza para fabricar piezas de automóviles. Juguetes: Se utiliza para fabricar juguetes. Envases de alimentos: Se utiliza para fabricar recipientes para un servicio de alimentos. Sistemas de amortiguación: Se utiliza para fabricar sistemas de amortiguación para el envío de productos electrónicos delicados. Producción de películas: Se utiliza para fabricar películas, ya que es transparente, duradero e imprimible. Envasado de alimentos: Se utiliza para envasar alimentos, ya que ayuda a preservar las vitaminas. El poliestireno es uno de los plásticos más comunes y se utiliza en formas sólidas, películas y espumas. El poliestireno expandido se utiliza en una amplia gama de aplicaciones, como aislamiento térmico y acústico en la construcción, embalaje de protección para productos frágiles, flotadores para la industria marina, piezas de automóviles, juguetes, y muchas otras aplicaciones industriales y de consumo.
