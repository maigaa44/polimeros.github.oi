APLICACIONES DEL LDPE

El polietileno de baja densidad (LDPE) tiene muchas aplicaciones, entre ellas: Embalaje Se utiliza para fabricar bolsas de plástico, sacos de basura, botellas, tapas de recipientes y envases de alimentos. Tubos y tuberías Se utiliza en sistemas de riego para transportar agua potable al ganado y para regar cultivos. Componentes electrónicos Se utiliza para fabricar piezas de plástico para componentes informáticos. Equipos de laboratorio Se utiliza para fabricar diversos equipos de laboratorio moldeados. Superficies de trabajo Se utiliza para fabricar superficies de trabajo resistentes a la corrosión. Film transparente Se utiliza para elaborar film transparente, tanto doméstico como industrial. El LDPE es un polímero termoplástico con propiedades únicas que lo hacen ideal para una gran variedad de aplicaciones. Entre sus propiedades se encuentran: Alta resistencia al impacto a bajas temperaturas Buena resistencia a la intemperie Buena resistencia química a alcoholes, álcalis diluidos y ácidos Excelentes propiedades aislantes eléctricas Muy baja absorción de agua
